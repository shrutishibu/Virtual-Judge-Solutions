Write a program which prints 1000 "Hello World".

Input
There is no input for this problem.

Output
The output consists of 1000 lines. Print "Hello World" in each line. 


//SOLUTION
#include<bits/stdc++.h>
using namespace std;
int main(){
    int i = 1000;
    while(i>0){
        cout<<"Hello World\n";
        i--;
    }
}
